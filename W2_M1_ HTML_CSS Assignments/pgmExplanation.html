Explanation of Carmine's Pizzeria Program

Introduction to Carmine's Pizzeria Program

The portion of Carmine's Pizzeria Program that I will be talking about is:

"Each of our pizzas has a volume of π·radius2·height. If radius = z & height = a, that means the volume spells "Pi·z·z·a"! "

Web Programming with HTML5, CSS, and JavaScript, by John Dean, Jones & Bartlett Learning, 2019, pp. 64–65.

Why it is interesting...

The reason this HTML code is interesting is because of how character references are used, how they can be incorporated into a source code, and when to use them.
For example, let us take a look at the section where Pi·z·z·a is displayed; prior to learning HTML, I had expected to use the letter x to signify multiplication and italics to represent variables such as z. We don't use those phrasing elements but rather we use their respective character references.

A lot of the work I perform as a Chemical and Process Engineer requires the utilization and usage of equations. So by learning and using character references, it makes the webpage look cleaner and easier to read. In my opinion, the addition of character references to web pages that have equations in them is very crucial as it allows the reader/user to fully grasp what you, as the programmer/author, are trying to convey.

Additionally, the proper use of HTML elements allows web search engines such as Google © to accurately read and analyze the content of the webpages as to provide the right webpage to the end-user. It's not even only search engines that read the content of webpages, devices that utilize TTS features read the source code and the right phrasing elements need to be used so it can output the right symbol, context, tone, and or articulation.
